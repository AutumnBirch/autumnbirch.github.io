## 部署

第一步：下载或git clone这个仓库至本地。

```
git clone https://github.com/AutumnBirch/autumnbirch.github.io.git
```

第二步：进入根目录文件夹（正常来说是/autumnbirch.github.io），执行以下命令。

```
npm install
```

第三步：执行以下命令，即可在http://localhost:4321/ 当中查看(有时候会是4322或者其他的)。

```
npm run dev
```
