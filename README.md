## 部署

首先你需要下载或git clone这个仓库。

```
git clone https://github.com/AutumnBirch/autumnbirch.github.io.git
```

然后进入根目录文件夹下，执行以下命令

```
npm install
```

再执行以下命令，即可在http://localhost:4321/ 当中查看了。

```
npm run dev
```
